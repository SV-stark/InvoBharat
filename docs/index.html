Visual Walkthrough

Explore the Interface

InvoBharat features standard UI libraries compiled natively, giving responsive rendering and lightning speed.

Dashboard View (Windows Fluent Design)

Real-Time Financial Dashboard

Keep a pulse on your business metrics. Real-time widgets show your revenue trends, dynamic tax liabilities (split clearly into CGST, SGST, IGST), and aging unpaid invoices directly on startup. Filters allow tracking by custom ranges, financial quarters, or months.

  • Visual Charts: Built-in fl_chart displays revenue and tax liability curves instantly.
  • Safe SQLite backend: Direct local indexing yields instant query speeds even with thousands of invoices.

Compliant & Clean Editor

The billing dashboard automates tax configuration. Enter client GSTIN or Select Client to auto-populate GST details, check state code, extract PAN, and determine if CGST/SGST or IGST should apply.

  • Series Sequences: Maintain distinct invoicing streams (e.g., `INV-` for retail, `SRV-` for service contracts).
  • HSN/SAC Lookups: Built-in search directory of standard tax codes with automatic slab assignment.

Professional Printed Invoices

Generate highly-polished, customer-facing PDF sheets in standard A4 format. Customize layout features directly from the settings drawer, including letterhead configurations, custom signatures, notes, and terms conditions.

  • Dynamic UPI QR Code: Renders scan-and-pay block containing invoice value, allowing customers to pay instantly.
  • HSN Tax Summary: Generates GSTR-ready summary grids grouped by tax rates.

Android Companion App

Billing on the go. compiles the core Drift SQLite database and Riverpod controllers natively on Android. Utilizes Material Design 3 guidelines featuring dynamic colors that harmonize with your operating system theme.

  • Seamless Backup Sync: Sync local files using secure database imports/exports.
  • Android Share Sheets: Directly share invoice PDFs through WhatsApp, Telegram, or Gmail.
Feature Deep-Dive

Engineered for Modern Indian Businesses

InvoBharat packs powerful corporate tools inside a lightweight, standalone desktop client.

Multi-Profile Business Management

Run separate entities, freelancing profiles, or retail storefronts side-by-side. Each profile maintains independent clients, products, custom logos, GSTIN codes, sequences, and settings drawers.

Built-in HSN/SAC Directory

Lookup typical HSN code values inside the product manager instantly. The library automatically associates tax slabs (5%, 12%, 18%, 28%) to catalog records, accelerating item selection during checkout.

Aging Receivables Reports

Stay on top of liquidity. Visual analytics segment unpaid and partially paid balances into standard aging buckets (0-30 days, 31-60 days, 61-90 days, and 90+ days overdue) for quick follow-up.

Recurring Invoices

Configure templates for ongoing agreements or monthly subscriptions. Set recurrence metrics and receive warnings or auto-generate increments on scheduling dates.

SMTP Direct E-Mailing

Send generated PDFs as attachments directly through secure SMTP protocols. Set up custom connections with providers like Gmail and dispatch invoices without copying files to standalone clients.

Auto-Backup & Migration

Configure automatic scheduled backups (daily, weekly, monthly) to local drives. GSTR-1 CSV exports and historic JSON data importers allow migrating database records seamlessly.

cd InvoBharat

# 2. Get dependencies
flutter pub get
